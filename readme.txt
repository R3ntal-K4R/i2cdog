This code is to demonstrate the use of I2C to communicate between a Raspberry Pi, a Raspberry Pi Pico and an Arduino. 

For more details see:
http://www.penguintutor.com/electronics/i2c

pico-demo1 (C code and Python code for Raspberry Pi to Pico)
arduino-demo1 (C code and Python code for Raspberry Pi to Arduino)
both-demo (Python code to communicate with both Pico and Arduino)
